# 🎓 Smart Campus Management System (SCMS)

## 📌 About
Smart Campus Management System (SCMS) is a web-based application developed to manage various campus services such as attendance, library, food ordering, and student information through a single platform.

## 🚀 Features
- Student Login
- Dashboard
- Attendance Management
- Library Management
- Food Ordering
- About & Contact Pages

## 🛠 Technologies Used
- HTML
- CSS
- Python 
- CSV Files
- Git & GitHub
